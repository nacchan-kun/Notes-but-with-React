## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open your browser to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## How It Works

1. **Step 1:** Enter your decision name and the options you're considering
2. **Step 2:** List pros and cons for each option
3. **Step 3:** Answer 4 questions about your priorities
4. **Step 4:** Get a personalized recommendation with detailed analysis

## Tech Stack

- React 18
- Vite
- CSS3 (Cyberpunk theme)
- localStorage for data persistence
